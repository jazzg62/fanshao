# shaoma

**收银台客户端，基于electron。**

## 快速开始

克隆和运行这个项目你需要安装[Git](https://git-scm.com) 和 [Node.js](https://nodejs.org/en/download/) (带有 [npm](http://npmjs.com)的版本)，然后在命令行运行

```bash
# 克隆项目
git clone git@github.com:jazzg62/fanshao.git
# 安装依赖
npm install
# 运行项目
npm start
```
